# Tryimage2 Image Theme Process

更新时间：2026-05-04

## 任务规则

- 每 4 小时回到当前线程，先阅读本文件，再做最新热点/行业时事/知识体系/认知热点调研。
- 每轮先给出至少 3 个“以小见大”的主题选择，用户选中后再使用 `image-gallery-publisher` 生成、导入并发布图片。
- 已画主题、候选主题、生成经验和避重复点都记录在本文件。
- 选题优先满足：有现实热度、有知识纵深、可视觉化、能做成 15 张一组、和历史批次不重复。
- 自动化：Codex heartbeat `tryimage2`，每 4 小时唤醒当前线程继续本流程。
- 多样性约束：每轮候选池至少覆盖 5 个不同母题，AI/科技主题最多占三分之一；优先混合自然生态、城市生活、食物供应链、历史考古、艺术展览、社会制度、科学现场、日常物件。
- 系列性约束：候选主题必须能自然拆成 10-15 张“具体对象/场景/角色/步骤”，避免只有概念名词、抽象议题或同构画面。

避重复原则：

- 不再做泛泛“大模型架构/方法论”或“复杂系统原理”总论，除非切入具体新事件。
- 不再做单纯“集体记忆/哲学体系/非遗”百科式清单，后续要从具体器物、场景、制度或危机切入。
- 技术主题要落到可画的微观场景：机房、电表、药盒、课堂、月球观察窗、知识库节点等。
- 不再做泛“载人航天/机器人十五讲”；航天要从具体设备、科学载荷或风险管理切入，机器人要从具体行业流程切入。

- 科学新闻中也有“微观生命现场”的线索：2026 年春季报道涉及灰鲸进入旧金山湾、帝企鹅和南极 fur seal 风险、濒危鼠类适应气候变化等，但这些更适合后续做自然纪录片题，目前系列性略弱于水、地铁、废水和食物链。

候选主题：

1. 地铁站的一间泵房，怎样替城市挡住热浪和暴雨
   - 母题：城市生活 / 气候适应 / 基础设施。
   - 小切口：地铁站地下泵房和站台热暴露，而不是宏大的“气候危机”海报。
   - 大问题：老城市基础设施如何同时面对暴雨倒灌、轨道高温、电力脆弱、乘客避暑和预算优先级。
   - 可做 10-15 张：站口防洪门、地下泵房、轨道热胀巡检、信号机柜防水、站台喷雾降温、乘客避暑区、无障碍电梯沙袋、隧道排水沟、公交接驳点、夜间维修队、电力室、风险地图桌、站务员应急广播、改造前后剖面、雨后恢复首班车。
   - 避重复：延续第三轮未生成的城市气候方向，但切得更具体；和已有“不可能建筑”“复杂系统”不同，它是现实交通系统现场。

2. 一杯自来水里的隐形清单
   - 母题：公共卫生 / 环境化学 / 日常物件。
   - 小切口：厨房水杯、净水器滤芯、水厂检测瓶、实验室显微颗粒。
   - 大问题：微塑料、药物残留、PFAS 和消毒副产物如何进入饮用水治理，科学证据、监管清单和家庭焦虑如何相互推动。
   - 可做 10-15 张：清晨水杯、老旧水管、滤芯剖面、水厂取样瓶、显微镜下微塑料、药物残留检测、PFAS 采样、活性炭滤床、反渗透膜、实验室质谱、居民听证会、学校饮水机、河流上游排口、监管清单会议桌、家庭换滤芯。
   - 避重复：不同于已有生命科学机理图和 One Health 候选，本题从饮用水治理和家庭水杯进入，画面更日常、更制度化。

3. 一块巧克力为什么还没便宜
   - 母题：食物供应链 / 消费价格 / 气候农业。
   - 小切口：超市货架上一块没有降价的巧克力。
   - 大问题：期货价格、天气病害、库存、合同、关税、品牌定价和消费者心理如何让“原料跌价”不等于“零售价下跌”。
   - 可做 10-15 张：可可果、病害叶片、发酵木箱、晾晒场、港口麻袋、期货交易屏、巧克力工厂、配方替代实验、超市价签、 shrinkflation 包装、精品巧克力柜台、农民合作社、品牌定价会议、消费者账单、未来气候温室。
   - 避重复：区别于旧候选“一颗可可豆的气候旅程”，本轮强调价格传导和零售错觉；图库尚未有食物供应链批次。

4. 一座城市的麻疹废水预警地图
   - 母题：公共卫生 / 城市监测 / 数据如何变成行动。
   - 小切口：污水处理厂里一瓶样本和地图上的一个检测点。
   - 大问题：社区级风险如何从下水道样本变成医疗提醒、疫苗门诊、学校通知和地方公共卫生决策。
   - 可做 10-15 张：下水道采样、污水厂冷藏箱、实验室 PCR、地图检测点、公共卫生电话会、儿科诊所、疫苗流动站、学校护士办公室、社区公告栏、旅行病例追踪、假阴性讨论、数据延迟看板、地方卫生局夜班、媒体问答、检测解除后的空地图。
   - 避重复：与抗生素 One Health 候选同属公共卫生，但对象和视觉流程完全不同；不是病原体机理图，而是城市预警系统。

5. 被 AI 帮完以后，技能有没有真的长出来
   - 母题：认知科学 / 教育 / 日常学习。
   - 小切口：一个学生或程序员把练习交给 AI 后，自己是否还能独立完成下一题。
   - 大问题：效率、认知卸载、学习动机、交互质量和技能形成之间的真实取舍。
   - 可做 10-15 张：AI 草稿、空白练习本、代码练习、老师批注、引用核验卡、反问清单、延迟测试、错题复盘、同伴讨论、学习动机仪表、关闭 AI 的十分钟、重新亲手写、能力体检、课程设计会议、长期技能曲线。
   - 避重复：这是本轮唯一 AI/认知候选；和已有大模型方法论、GraphRAG、AI 教育候选不同，重点在“技能是否形成”的实验现场和学习动作。

## 2026-05-04 第三轮候选池

- 改进规则：本轮重做为跨领域候选池，AI/科技只保留为少数；每个主题都给出 12-15 个可画画面，确保系列感来自对象、地点、流程或角色变化，而不是同一概念反复换皮。

1. 地铁站的一份气候韧性改造清单
